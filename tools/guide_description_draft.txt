-EN-(DeepL Translate)

This is a collection of recommended addons for playing GmodVR.
This guide uses "VRMod:SemiOfficialVersion".
(There is also a fork called "VRMod x64", but this guide does not cover it.)


● How to Install ●

Please refer to the image shown above to install the software.

1) Download the VRMod runtime from here
https://steamcommunity.com/sharedfiles/filedetails/?id=1678408548

2) Open the downloaded Zip file and select "vrmod-module-master"->"install"->"GarrysMod" in this order.

3) Right-click "Garry'smod" in the Steam library and click "Manage"->"Browse Local Files" to open the location of Garry'smod.

4) Overwrite the three folders "bin", "garrysmod", and "vrmod_installer" from the zip into the Garry'smod folder opened in step (3).

5) Then press the "Subscribe All" button in this collection to start Gmod.

6) Once started, it should work by typing [Start new game]→(select map)→F10→"vrmod".

7) If you want the vrmod menu to appear immediately after this, type "bind L "vrmod"" from the F10 console, and the next time you press the L key, the VRMod shortcut menu will appear.


● If it does not work properly ●

Please lower the image quality setting as much as possible and set the gmod resolution to 1366*768.


● If you get low FPS ●

[If you are using VRMod:SemiOfficialPatch]
After starting the game, enter "vrmod_gmod_optimization" from the F10 console to automatically adapt the settings for vrmod.
The ConVars listed below are automatically applied, so you don't need to do anything manually.

[If you are NOT using VRMod:SemiOfficialPatch, or if you want to set them manually]
Open "autoexec.cfg" at the following location and copy and paste the settings below.
[SteamLibrary\steamapps\common\GarrysMod\garrysmod\cfg\autoexec.cfg]

//--------------------------------------------
//① Turn on sv_cheats
sv_cheats 1

//② Shorten the map draw distance
r_mapextents 1650

//③ Turn off skybox
r_3dsky 0

//④ Disable water reflection
r_cheapwaterend 0
r_waterforcereflectentities 0
r_waterdrawreflection 0
r_waterdrawrefraction 0
r_waterforceexpensive 0

//⑤ Disable shadows
mat_shadowstate 0
r_shadowmaxrendered 0
r_shadowrendertotexture 0
r_shadows 0
r_threaded_client_shadow_manager 0
r_threaded_renderables 0

//⑥ Change Gmod settings that are incompatible with VRMod
studio_queue_mode 1
prop_active_gib_limit 50
prop_active_gib_max_fade_time 5
snd_mix_async 1
mat_parallaxmap 1
mat_queue_mode -1
dsp_enhance_stereo 0
dsp_slow_cpu 1
mat_vsync 0
mat_motion_blur_enabled 0
//--------------------------------------------


● If this is still too heavy ●

(This is just a command to verify the operation. This setting makes it impossible to see far away at all.
If this increases the FPS, try increasing r_mapextents a little bit to find the right distance for your PC.)

r_mapextents 400
r_3dsky 0
r_decals 5

-JP-

GmodVRをプレイするためのおすすめアドオンコレクションです。
このガイドでは「VRMod:SemiOfficialVersion」を使用しています。
（「VRMod x64」というフォークも存在しますが、このガイドでは扱いません。）


● インストール方法 ●

上に表示されている画像を参考にして導入してみてください。

①ここからVRModのランタイムをダウンロードする
https://steamcommunity.com/sharedfiles/filedetails/?id=1678408548

②ダウンロードしたZipを開き、「vrmod-module-master」→「install」→「GarrysMod」の順に開く

③Steamのライブラリから「Garry'sMod」を右クリックして「管理」→「ローカルファイルを閲覧」を押してGarry'sModの場所を開く

④zipの中から「bin」「garrysmod」「vrmod_installer」の3つのフォルダを③で開いたGarry'sModのフォルダに上書きする

⑤その後にこのコレクションの「すべてをサブスクライブ」ボタンを押してGmodを起動する

⑥起動したら、[新しいゲームを開始]→(マップを選択)→F10→「vrmod」と入力すれば動くと思います

⑦もしvrmodメニューをすぐに出したい場合は、F10のコンソールから「bind L "vrmod"」と入力しておけば、次からはLキーを押すとVRModのショートカットメニューが出現します


● 正常に動作しない場合 ●

画質設定をできるだけ下げて、Gmodの解像度は1366×768にしてください。


● 動作が重い場合 ●

【VRMod:SemiOfficialPatchをご使用の方】
ゲームを起動したら、F10のコンソールから「vrmod_gmod_optimization」と入力すれば、vrmod向けの設定が自動的に適用されます。
下記のConVarは自動で適用されるため、手動で設定する必要はありません。

【VRMod:SemiOfficialPatchを使っていない方、または手動で設定したい方】
下記の場所にある「autoexec.cfg」を開き、以下の設定をコピー＆ペーストしてください。
[SteamLibrary\steamapps\common\GarrysMod\garrysmod\cfg\autoexec.cfg]

//--------------------------------------------
//①sv_cheatsをON
sv_cheats 1

//②マップの描画距離を短くする
r_mapextents 1650

//③skyboxを切る
r_3dsky 0

//④水の反射を無効にする
r_cheapwaterend 0
r_waterforcereflectentities 0
r_waterdrawreflection 0
r_waterdrawrefraction 0
r_waterforceexpensive 0

//⑤影を無効化する
mat_shadowstate 0
r_shadowmaxrendered 0
r_shadowrendertotexture 0
r_shadows 0
r_threaded_client_shadow_manager 0
r_threaded_renderables 0

//⑥VRModと相性の悪いGmodの設定を変更する
studio_queue_mode 1
prop_active_gib_limit 50
prop_active_gib_max_fade_time 5
snd_mix_async 1
mat_parallaxmap 1
mat_queue_mode -1
dsp_enhance_stereo 0
dsp_slow_cpu 1
mat_vsync 0
mat_motion_blur_enabled 0
//--------------------------------------------


● これでも重い場合 ●

（これはあくまで動作検証用のコマンドです。この設定は遠くが全く見えなくなります。
もしこれでFPSが上昇した場合はr_mapextentsを少しずつ上げて、PCに合った描画距離を探してみてください。）

r_mapextents 400
r_3dsky 0
r_decals 5
